Hygpup

CPSC 599 Final Project

The goal of Hygpup is to strengthen the bond between you and your pet. Hygpup does so by reminding you when it’s time to show your pet some love through the use of notification alerts. Hardware design consists of an ESP8622 Node MCU module for establishing a network connection and an ADXL 345 accelerometer for inactivity detection.

The inspiration behind Hygpup was a situation that happened to me right after quarantine period started. As a result of doing all the work from my room, I was constantly busy working. When getting out of my room to eat, I’ve observed my pets sitting on staircase and asking for belly rubs from anyone passing by. This has inspired me to create a wearable device that would alert me of my pets’ inactivity in order to find the best time to spend some time with my fluffy friends.

Implementation has been done with the help of my dogs, Loki and Eva, which is what makes this project unique for me!

Circuit Schematic

The diagram has been created using Fritzing application and outlines sensor connections.

Hardware Assembly #1:

The initial version of hardware used to also include an Arduino Nano. The ESP8622 Node MCU module used to communicate with Arduino Nano in order to send data read from the accelerometer.

Hardware Assembly #2:

I was able to optimize the use of space later on in the project by removing the need to use Arduino Nano. As a result, accelerometer was directly connected to ESP8622 Node MCU module which also reduced the complexity of the circuit significantly.

Embodiment #1:

With the help of my grandma, I managed to turn my old shorts into a small cape for puppies. There are 4 LEDs attached to the cape in order to communicate the state of sensors.

Embodiment #2:

I've also sewn a small pocket in order to hold the hardware.

Finalized Project

Finally, I was able to test functionality with the help of Eva.
